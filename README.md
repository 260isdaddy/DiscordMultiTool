# GostMultiTool - A Discord Automated Bot Solution

### Features

 - Runescape item price search
 - Steam game search

### Requirements
 - Python3
 - The key has to be provided in a **DiscordBotKey.py** file like this:

		private_Key("key here") 


### Run
To start: In **./venv** folder run **BotMain.py** file.
